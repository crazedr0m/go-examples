main:
прочитать параметры, обработать конфиг
зарегистрировать обработчик сигналов
запустить логирование и мониторинг
запустить сервис обработки сообщений

сервис обработки сообщений:
создать минимальный пулл обработчиков
создать подключение к брокеру сообщений
зарегистроировать обработчик сообщений

обработчик сообщений:
из сообщения получить id задачи
уведомить что сообщение обработано
получить данные по id задачи.
обновить статус здачи на "в работе"
запустить в отдельном потоке выполнение задачи через системный выззов
ждать завершения задачи.
